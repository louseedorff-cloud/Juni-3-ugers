# Juni-3-ugers
In this GitHub repository, we will explain with a few words how to navigate our files.

File: datasæt_12cv.pdf. This contains the CV's used in this project. 

File: Bias_study_CHATGPT.ipynb. This Jupiter Notebook contains the entire statistical analysis for the LLM Chat GPT.  

File: Bias_study_GROK.ipynb. This Jupiter Notebook contains the entire statistical analysis for the LLM GROK. 

File: Effectsize_pilot_chatgpt.ipynb. In This Jupiter Notebook, are the calculations for a pilot-study for chatgpt, to determine the effectsize. 

File: Effectsize_pilot_grok.ipynb. In This Jupiter Notebook, are the calculations for a pilot-study for grok, to determine the effectsize. 
The effectsizes were then used with the software G*Power, to determine the sample sizes for the original study for the LLM. 


File: Comparing_GROK_Chatgpt.ipynb. This Jupiter Notebook regards the comparison for Chat GPT and Grok.
